# Список рекомендованного оборудования

## Полный список элементов для генерации изображений

### 1. Шейкер
- **Название:** Шейкер
- **Подзаголовок:** Boston или Cobbler
- **Описание:** Основной инструмент для смешивания коктейлей. Бывает двух типов - Boston (две части) и Cobbler (с встроенным ситечком)
- **Файл изображения:** `shaker.jpg` или `shaker.png`

---

### 2. Барная ложка
- **Название:** Барная ложка
- **Подзаголовок:** Для перемешивания
- **Описание:** Длинная спиральная ложка для аккуратного перемешивания коктейлей
- **Файл изображения:** `bar-spoon.jpg` или `bar-spoon.png`

---

### 3. Стрейнер
- **Название:** Стрейнер
- **Подзаголовок:** Ситечко
- **Описание:** Специальное ситечко для процеживания коктейлей, удержания льда и фруктов
- **Файл изображения:** `strainer.jpg` или `strainer.png`

---

### 4. Джиггер
- **Название:** Джиггер
- **Подзаголовок:** Мерный стаканчик
- **Описание:** Двусторонний мерный стаканчик для точного отмеривания ингредиентов (обычно 30мл/60мл)
- **Файл изображения:** `jigger.jpg` или `jigger.png`

---

### 5. Мадлер
- **Название:** Мадлер
- **Подзаголовок:** Пестик
- **Описание:** Деревянный или пластиковый пестик для измельчения фруктов, трав и специй
- **Файл изображения:** `muddler.jpg` или `muddler.png`

---

### 6. Стаканы
- **Название:** Стаканы
- **Подзаголовок:** Различных типов
- **Описание:** Набор различных типов стаканов: рокс, хайбол, мартинка, купе и другие
- **Файл изображения:** `glasses.jpg` или `glasses.png`

---

## Инструкции по размещению изображений

1. Сгенерируйте или найдите качественные изображения для каждого элемента
2. Размер: минимум 400x400px для оптимального отображения
3. Формат: JPG или PNG с прозрачностью
4. Поместите изображения в эту папку (`client/public/equipment/`)
5. Используйте предложенные названия файлов или обновите код в `courses.tsx`

## Обновление кода

После добавления изображений, обновите массив `equipment` в файле `client/src/pages/courses.tsx`:

```typescript
const equipment = [
  {
    name: "Шейкер",
    subtitle: "Boston или Cobbler",
    image: "/equipment/shaker.jpg"  // ← обновите путь
  },
  // ... остальные элементы
];
```
