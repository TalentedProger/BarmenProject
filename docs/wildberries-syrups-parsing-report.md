- **category** - `"syrup"`

- **unit** - `"ml"`
- **sourceUrl** - ссылка на товар на Wildberries
- **sourceName** - `"Wildberries"`
- **sourceIcon** - иконка Wildberries (URL)
- **imageUrl** - ссылка на фотографию товара (первое изображение)
