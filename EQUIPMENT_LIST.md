# Рекомендованное оборудование для курсов барменов

## Список из courses.tsx

1. **Шейкер**
   - Подтип: Boston или Cobbler
   - Использование: Смешивание коктейлей встряхиванием

2. **Барная ложка**
   - Подтип: Для перемешивания
   - Использование: Перемешивание коктейлей, создание слоёв

3. **Стрейнер**
   - Подтип: Ситечко
   - Использование: Фильтрация льда и мякоти при наливании

4. **Джиггер**
   - Подтип: Мерный стаканчик
   - Использование: Точное измерение ингредиентов

5. **Мадлер**
   - Подтип: Пестик
   - Использование: Растирание фруктов, трав и специй

6. **Стаканы**
   - Подтип: Различных типов
   - Использование: Подача коктейлей (хайбол, рокс, мартини, купе и др.)

---

## Примечание
*Базовый набор для начала обучения. Дополнительное оборудование понадобится на продвинутых курсах*

## Дополнительное оборудование (продвинутый уровень)

- Блендер для смузи и фраппе
- Пресс для цитрусовых
- Нож для нарезки гарниров
- Разделочная доска
- Поддон для бара
- Щипцы для льда
- Дозаторы для сиропов
- Миксер для молочных коктейлей
