# Инструкция по ручному сбору изображений для соков

## Паттерн URL изображения
```
https://krasnoeibeloe.servicecdn.ru/upload/resize_cache/iblock/[ID]/[HASH]/200_356_1/[filename].jpg
```

## Пример
**Товар:** Напиток сокосодержащий Добрый Апельсин мандарин 2 л  
**URL изображения:** https://krasnoeibeloe.servicecdn.ru/upload/resize_cache/iblock/f5f/een22d3en21bxrii0xefa5vfv2dm2los/200_356_1/mandarin_040624_t.jpg

## Как получить URL изображения:

### Способ 1: Из страницы продукта
1. Откройте sourceUrl товара в браузере
2. Откройте DevTools (F12)
3. Найдите элемент `<img>` с изображением товара
4. Скопируйте значение атрибута `src`
5. Используйте это значение как `imageUrl`

### Способ 2: Из каталога
1. Откройте https://krasnoeibeloe.ru/catalog/soki-i-nektary/
2. Примените фильтры (соки, нектары, напитки, объем 0.2-2л)
3. Найдите нужный товар
4. Правый клик на изображение → "Копировать адрес изображения"

## Инструкция по обновлению файлов

После сбора URL нужно добавить их в соответствующий файл:

```typescript
{ 
  name: "Нектар мультифрукт 1.93л", 
  category: "juice", 
  // ... другие поля
  imageUrl: "https://krasnoeibeloe.servicecdn.ru/upload/resize_cache/iblock/XXX/YYY/200_356_1/filename.jpg"
}
```
